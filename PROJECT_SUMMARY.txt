📡 ENDPOINTS DISPONIBLES
═════════════════════════════════════════════════════════════════════════════

1. GET /health
   ├─ Descripción: Health check
   ├─ Respuesta: {"status": "healthy", "message": "..."}
   └─ Status: 200

2. POST /predict
   ├─ Descripción: Realizar predicción
   ├─ Input JSON:
   │  ├─ service_date (date): Fecha a predecir
   │  ├─ max_temp_c (float): Temperatura Celsius
   │  ├─ precipitation_mm (float): Precipitación mm
   │  ├─ is_stadium_event (bool): ¿Hay evento?
   │  └─ is_payday_week (bool): ¿Semana cobro?
   ├─ Respuesta JSON:
   │  ├─ prediction_result (int): Resultado IA
   │  ├─ service_date (date): La fecha predicha
   │  ├─ model_version (str): Versión del modelo
   │  ├─ execution_timestamp (datetime): Timestamp
   │  └─ log_id (int): ID del registro en BD
   ├─ Status: 201 (Created)
   └─ Error handling:
      ├─ 422: Datos inválidos (validación Pydantic)
      ├─ 503: Motor no disponible
      └─ 500: Error interno con rollback

3. GET /docs
   └─ Swagger interactivo (auto-generado)

✨ FLUJO COMPLETO
═════════════════════════════════════════════════════════════════════════════

Cliente HTTP
    ↓
POST /predict en JSON
    ↓
Pydantic valida request
    ↓
FastAPI inyecta sesión DB
    ↓
PredictionEngine.predict('model', data)
    ↓ (retorna predicción)
    ↓
Crea PredictionLog ORM
    ↓
db.add() + db.commit()
    ↓
INSERT en tabla PredictionLogs
    ↓
Retorna JSON response
    ↓
Cliente HTTP (log_id incluido)

🎓 EJEMPLOS DE USO
═════════════════════════════════════════════════════════════════════════════

CURL - GET /health:
   curl http://localhost:8000/health

CURL - POST /predict:
   curl -X POST http://localhost:8000/predict \
     -H "Content-Type: application/json" \
     -d '{
       "service_date": "2026-03-15",
       "max_temp_c": 28.5,
       "precipitation_mm": 0.0,
       "is_stadium_event": false,
       "is_payday_week": true
     }'

PowerShell - POST /predict:
   $body = @{
       service_date = "2026-03-15"
       max_temp_c = 28.5
       precipitation_mm = 0.0
       is_stadium_event = $false
       is_payday_week = $true
   } | ConvertTo-Json
   
   Invoke-RestMethod -Uri "http://localhost:8000/predict" `
     -Method POST `
     -Headers @{"Content-Type"="application/json"} `
     -Body $body

PYTHON - Cliente de la API:
   import requests
   
   response = requests.post(
       "http://localhost:8000/predict",
       json={
           "service_date": "2026-03-15",
           "max_temp_c": 28.5,
           "precipitation_mm": 0.0,
           "is_stadium_event": False,
           "is_payday_week": True
       }
   )
   print(response.json())
